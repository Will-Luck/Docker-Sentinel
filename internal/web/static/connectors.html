Connectors

External service integrations

Portainer Instances

Connect to one or more Portainer instances to monitor and update containers on remote Docker hosts. Each instance can have multiple endpoints; local socket endpoints are auto-detected and blocked to avoid duplicating what Sentinel monitors directly.

Nginx Proxy Manager

Connect to NPM to auto-discover reverse proxy domains. Port chips on the dashboard will link to the proxy domain instead of the raw host:port.

Portainer Integration

Connect to a Portainer instance to view endpoints and containers from the Portainer dashboard.

Enable Portainer
Show the Portainer dashboard and scan Portainer-managed containers. Requires a restart after changing URL or token.
Portainer URL
Base URL of your Portainer instance, e.g. https://portainer.example.com
API Token
Portainer API access token (Settings › Users › Access tokens).
Connection
Test connectivity with the current URL and token.

Nginx Proxy Manager

Connect to NPM to auto-discover reverse proxy domains. Port chips on the dashboard will link to the proxy domain instead of the raw host:port.

Enable NPM
Auto-resolve container ports to NPM proxy domains. Requires a restart after changing credentials.
NPM URL
Admin UI URL of your NPM instance, e.g. http://192.168.1.57:62453
Email
NPM admin email address for API authentication.
Password
NPM admin password.
Credentials
Save email and password.
Connection
Test connectivity with the current URL and credentials.
Sync
Force a re-sync of proxy host mappings from NPM.
